HTML

(Hyper Text Markup Lenguaje)


Es de los primeros lenguajes para paginas web, solo sirve para indicar como va ordenado el contenido de una pagina web. Esto lo hace por medio de etiquetas o conocidas en ingles como tags.


En este se enlazan los estilos que llevara la pagina web y la interactividad con el usuario, la cual se hace por medio de javascript.
