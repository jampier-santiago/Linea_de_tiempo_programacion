Javascript

Es un lenguaje de programación interpretado.


Este lenguaje permite implementar funciones complejas en una pagina web. Una pagina estatica, que simplemente se encarga de mostrar informacion, en la mayoria de los casos no posee javascript.


Javascript muestra actualizaciones de contenido, animaciones, mapas intercativos, hace operaciones, puede validar datos y ayudar con algunas busquedas, esto logra que una pagina posea mas interactividad entre el usuario y nuestra creación.
